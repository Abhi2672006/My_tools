# Bash TCP Port Scanner

A simple TCP port scanner written in pure Bash using the `/dev/tcp`

## Purpose
This project is part of my ethical hacking learning journey.
The script scans ports **1-1024** and prints open ports.

## Requirements
- Linux / MacOS
- Bash shell
- Internet / LAN target 

# Usage 

```bash
chmod +x bash-port-scanner.sh
./bash-port-scanner.sh <target-ip>
